# 火山引擎大模型流式语音识别配置指南

## 🎯 获取API凭证步骤

### 步骤1：登录火山引擎控制台

1. 访问：https://console.volcengine.com/
2. 使用您的账户登录

### 步骤2：开通大模型流式语音识别服务

1. 在控制台左侧导航中找到**"语音技术"**
2. 进入**"大模型流式语音识别"**服务页面
3. 如果服务未开通，点击**"立即开通"**
4. 确认服务已开通并有可用额度

### 步骤3：获取应用凭证

1. 在语音技术服务页面，找到**"应用管理"**
2. 如果没有应用，点击**"创建应用"**
3. 在应用列表中找到您的应用
4. 获取以下信息：
   - **APP ID**（应用ID）
   - **Access Token**（访问令牌）

### 步骤4：配置环境变量

将获取的信息填入 `.env` 文件：

```env
# 火山引擎大模型流式语音识别配置
VOLCENGINE_APP_KEY=您的APP_ID
VOLCENGINE_ACCESS_TOKEN=您的ACCESS_TOKEN
VOLCENGINE_ASR_RESOURCE_ID=volc.bigasr.sauc.duration
```

## 🚨 常见问题解决

### 问题1：401 Unauthorized

**原因：**
- APP_KEY 或 ACCESS_TOKEN 不正确
- 服务未开通或权限不足
- 账户余额不足

**解决方案：**
1. 重新检查控制台中的凭证信息
2. 确认服务已正确开通
3. 检查账户余额和配额

### 问题2：服务未开通

**原因：**
- 账户未开通大模型流式语音识别服务
- 服务地区限制

**解决方案：**
1. 在控制台中开通对应服务
2. 确认服务在您的地区可用

### 问题3：配额不足

**原因：**
- 免费配额已用完
- 付费配额不足

**解决方案：**
1. 检查控制台中的用量统计
2. 购买更多配额或升级套餐

## 🎯 Resource ID 说明

根据您的使用场景选择正确的Resource ID：

- **小时版**：`volc.bigasr.sauc.duration`
- **并发版**：`volc.bigasr.sauc.concurrent`

大多数情况下使用小时版即可。
