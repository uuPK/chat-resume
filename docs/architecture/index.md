# 架构文档入口

本页用于说明仓库内与系统结构相关的文档入口，目的是让智能体快速找到正确层级的说明。

## 核心入口

- `../../ARCHITECTURE.md`：系统分层、模块职责、数据流和依赖规则
- `project-structure.md`：目录总览和关键代码入口
- `current-test-report.md`：测试覆盖和当前验证现状

## 使用建议

- 要理解依赖边界时，优先读 `../../ARCHITECTURE.md`
- 要寻找文件位置和页面入口时，优先读 `project-structure.md`
- 要确认当前自动化验证能力和风险时，优先读 `current-test-report.md`
- 如果新增了模块级架构文档，也从本页继续挂载
