# Triage Labels

工程技能使用五个标准分诊角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的角色 | 本仓库使用的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待 reporter 补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，AFK agent 可以直接接手 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时，例如“apply the AFK-ready triage label”，请使用上表中对应的实际标签字符串。

如果以后仓库采用不同标签体系，只需要修改右侧“本仓库使用的标签”这一列。
